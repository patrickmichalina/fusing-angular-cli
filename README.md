## Prerequisites

The CLI has dependencies that require Node 10.0.0 or higher, together with NPM 6.0.0 or higher.

## Installation

```bash
npm install -g fusing-angular-cli
```

## Usage

```bash
fng help
```

### Updating Fusing Angular CLI

```bash
npm i -g fusing-angular-cli@latest
```
